## Overview

This is a simple budgeting application made with Flask and React, meant to help me gain experience with Restful APIs!

## Features

- **Add categories for spending type:** Add custom categories for your spending habits
- **Add purchases sorted by date:** Each purchase is automatically sorted by date and category
- **Overspending highlighted in red:** If a purchase exceeds the category spending goal, it will be highlighted in red to make it standout

A simple school assignment that taught me alot!
